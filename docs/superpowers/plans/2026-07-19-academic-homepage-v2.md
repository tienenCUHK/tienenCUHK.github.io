# Academic Personal Homepage v2 — Implementation Plan

> **Approach:** Pure static HTML/CSS inspired by Odin Zhang's page, using Skeleton CSS framework.

**Goal:** Build a single-page academic homepage with sections (Bio, News, Publications, Projects, Blog, CV) deployed to `tienenCUHK.github.io`.

**Tech Stack:** HTML5 + Skeleton CSS + minimal custom CSS + Font Awesome icons

## File Structure

```
.
├── index.html                    # Main single-page site
├── assets/
│   ├── img/
│   │   ├── prof_pic.jpg          # Profile photo (~400x400)
│   │   └── projects/             # Project screenshots
│   └── pdf/
│       └── cv.pdf                # CV download
├── libs/
│   ├── external/
│   │   ├── skeleton/             # Skeleton CSS framework
│   │   └── font-awesome/         # Icons
│   └── custom/
│       └── my_css.css            # Custom styles
├── blog/
│   └── index.html                # Blog listing page
├── blog/posts/                   # Individual blog posts
│   └── 2026-07-19-welcome.html
├── favicon.ico
└── README.md
```
